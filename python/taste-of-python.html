A Taste of Python

Welcome to the world of Python.

Python is Simple

Python is a very simple programming language and it is quite easy to learn.

Writing a hello-world program just takes a single line of code.

print("hello, world!")
hello, world!

It is very handy to use Python as a calculator.

print(1 + 2)

Python is Dynamically Typed

Python is a dynamically typed programming language, so you donâ€™t need to declare type of variables.

y = 2

It is perfectly fine to reassign a variable to a value of a different type.

x = 1
print(x)

x = "foo"
print(x)

foo

While the variables do not have types associated with them, the values do have types. Python is strict about them and it doesnâ€™t allow operations on incompatible datatypes.

Python uses Indentation

Python uses indentation to identify the code that is part of a block.

Notice that Python doesnâ€™t use the usual { and } characters to identify code blocks. It just uses indentation to identify the block of code that is part of compound statements like if, else, etc.
